Angular2 LifeCycle Hooks

Like our normal programming language concept of the life cycle, angular2 component has its own life cycle every component follow these life cycles hooks a number of different phases it goes through from birth to death. using these hooks we can control our component. To do this we add some specific methods to our component class which get called during each of these life-cycle phases, we call those methods hooks. hooks for component:- ngOnChanges() ngOnInit() ngDoCheck() ngOnDestroy()
